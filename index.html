Getting Started

Aether Terminal is a modern, AI-powered terminal for Windows that enhances your command-line experience.

System Requirements

  • Windows 10 or Windows 11
  • 4GB RAM minimum (8GB recommended)
  • 500MB free disk space
  • Internet connection for AI features
Quick Installation
winget install AetherApp.Terminal

Installation
